# Run Guide
1 open directory
```sh 
cd Proxy-Checker
```

2 edit file ( optional )
```sh
nano proxies.txt
```
paste your proxy here. if you want check manual and ```CTRL+X``` ```Y``` ```ENTER``` to save

3 run
```sh
python3 wxd4mode.py
```
4 Select Mode
  - 1 Grab and Checker Automatic
  - 2 Manual from file proxies.txt
  - 3 Manual from file (write your file name)
  - 4 Manual 1 proxy
